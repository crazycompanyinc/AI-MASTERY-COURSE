# Benchmarks and Datasets

## LLM Benchmarks
- MMLU: Massive Multitask Language Understanding
- HumanEval: Code generation
- GSM8K: Math reasoning
- HellaSwag: Commonsense reasoning
- TruthfulQA: Truthfulness
- MT-Bench: Conversation quality
- Chatbot Arena: Human preference

## Vision Benchmarks
- ImageNet: Image classification
- COCO: Object detection, segmentation
- Cityscapes: Urban scene understanding

## NLP Benchmarks
- GLUE/SuperGLUE: Language understanding
- SQuAD: Question answering
- WMT: Machine translation

## Datasets
- Common Crawl: Web text
- The Pile: Diverse text
- LAION: Image-text pairs
- OpenWebText: Reddit text
- CodeParrot: Code
