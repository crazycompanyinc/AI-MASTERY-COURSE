# Automation

## Scripts
- `daily_knowledge_expansion.py` — Expands course knowledge
- `daily_news_update.py` — Updates AI news and trends
- `run_coverage_audit.py` — Audits coverage

## Usage
```bash
python automation/daily_knowledge_expansion.py
python automation/daily_news_update.py
python automation/run_coverage_audit.py
```

## With git
```bash
python automation/daily_knowledge_expansion.py --commit --push
```
