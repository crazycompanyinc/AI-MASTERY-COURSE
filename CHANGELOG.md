### Automation
- `make install` — Install dependencies
- `make validate` — Validate course structure
- `make audit` — Run coverage audit
- `make update-knowledge` — Daily knowledge expansion
- `make update-news` — Daily news update
- `make update-all` — All updates
- `make website` — Build documentation website
- `make check-links` — Check for broken links

### Metrics
- AI_LIBRARY_TOTALITY_SCORE: computed by coverage audit
- Current score: computed at first run
