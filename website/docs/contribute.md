# Contribuir

## Cómo contribuir

1. **Fork** el repositorio
2. **Crea una rama**: `git checkout -b feature/mi-cambio`
3. **Haz cambios** siguiendo las guías
4. **Valida**: `make validate`
5. **Commit** y **Push**
6. **Pull Request**

## Guías
- Contenido original o citado
- Markdown para docs
- Python 3.10+ para código
- Español como idioma principal

Ver [CONTRIBUTING.md](../CONTRIBUTING.md) para detalles.
