# Glosario

Ver [glossary/global-glossary.md](../glossary/global-glossary.md) para el glosario completo.

## Términos principales

- **AGI**: Inteligencia Artificial General
- **Attention**: Mecanismo de atención
- **Backpropagation**: Retropropagación
- **CNN**: Red Neuronal Convolucional
- **Deep Learning**: Aprendizaje profundo
- **Embedding**: Representación vectorial
- **GAN**: Red Generativa Adversaria
- **LLM**: Modelo de Lenguaje Grande
- **ML**: Machine Learning
- **NLP**: Procesamiento de Lenguaje Natural
- **RAG**: Retrieval-Augmented Generation
- **RLHF**: Reinforcement Learning from Human Feedback
- **SGD**: Stochastic Gradient Descent
- **Transformer**: Arquitectura basada en atención
