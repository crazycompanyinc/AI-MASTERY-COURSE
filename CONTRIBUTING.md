# Contributing to AI-MASTERY-COURSE

¡Gracias por tu interés en contribuir! Este proyecto es open source y las contribuciones son bienvenidas.

## Cómo contribuir

### 1. Reportar errores
Abre un [issue](https://github.com/YOUR_USERNAME/AI-MASTERY-COURSE/issues) con:
- Descripción del error
- Pasos para reproducir
- Comportamiento esperado vs actual
- Capturas de pantalla si aplica

### 2. Solicitar contenido
Usa el template de [content request](https://github.com/YOUR_USERNAME/AI-MASTERY-COURSE/issues/new?template=content_request.md) para:
- Pedir nuevos módulos
- Solicitar más profundidad en un tema
- Sugerir nuevos laboratorios o proyectos

### 3. Contenido directo
Para contribuir contenido directamente:

1. **Fork** el repositorio
2. **Crea una rama**: `git checkout -b feature/mi-contribucion`
3. **Haz tus cambios** siguiendo las guías de contenido
4. **Valida**: `make validate`
5. **Commit**: `git commit -m "Descripción clara del cambio"`
6. **Push**: `git push origin feature/mi-contribucion`
7. **Abre un Pull Request**

## Guías de contenido

### Lecciones
- Usa el [template de lección](../templates/lesson-template.md)
- Incluye objetivos, explicación, conceptos clave, ejemplos, ejercicios
- Cita fuentes con enlaces
- Escribe en español (o inglés con traducción)
- No copies contenido protegido

### Laboratorios
- Código ejecutable y probado
- README con instrucciones claras
- requirements.txt si hay dependencias
- explanation.md con la teoría

### Proyectos
- Usa el [template de proyecto](../templates/project-template.md)
- Incluye arquitectura, implementación, evaluación
- Código funcional y documentado

## Estilo

- **Markdown** para toda la documentación
- **Python 3.10+** para código
- **Español** como idioma principal (inglés en /translations/en/)
- **Citas** para todas las fuentes
- **Original** — no copies contenido existente
