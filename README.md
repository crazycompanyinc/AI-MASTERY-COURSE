## Visión

AI-MASTERY-COURSE es una **bibliotiva viva** de Inteligencia Artificial que cubre, de forma progresiva y verificable, todo el conocimiento público relevante sobre IA. Funciona simultáneamente como:

1. **Curso completo** — De cero a investigación frontier
2. **Enciclopedia de IA** — Referencia permanente
3. **Biblioteca de papers** — 250+ papers catalogados y resumidos
4. **Laboratorio práctico** — 30+ labs con código ejecutable
5. **Colección de proyectos** — 30+ proyectos por nivel
6. **Website documental** — Navegable con MkDocs Material
7. **Sistema de actualización automática** — GitHub Actions + cron diario
8. **Sistema de auditoría de cobertura** — Métricas de completitud
9. **Base de conocimiento extensible** — JSON memory + grafo de temas
10. **Repositorio open source mantenible** — Contribuciones bienvenidas

## Instalación

```bash
# Clonar repositorio
git clone https://github.com/YOUR_USERNAME/AI-MASTERY-COURSE.git
cd AI-MASTERY-COURSE

# Crear entorno virtual
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows

# Instalar dependencias
pip install -r requirements.txt

# Verificar instalación
make validate
```

## Cómo usar el curso

### Estudio autodirigido
1. Elige tu ruta de aprendizaje arriba
2. Lee cada módulo en orden
3. Completa los ejercicios
4. Ejecuta los laboratorios
5. Construye los proyectos

### Laboratorios prácticos
```bash
cd labs/python-basics
pip install -r requirements.txt
python main.py
```

### Website documental
```bash
make website
# Abre http://localhost:8000
```

### Actualizaciones diarias
```bash
# Manual
make update-all

# Con cron (Linux/Mac)
crontab automation/cron.example
# Edita primero la ruta absoluta en cron.example
```
