# Política de Citación

## Formato
```markdown
Autor(es) (Año). Título. Fuente. URL
```

## Ejemplo
```markdown
Vaswani, A. et al. (2017). Attention Is All You Need. NeurIPS.
https://arxiv.org/abs/1706.03762
```

## Reglas
1. Citar toda fuente usada
2. Enlazar cuando sea posible
3. Marcar contenido pendiente de verificación
4. Diferenciar hechos, opiniones y especulación
