# Mantenimiento

## Tareas diarias (automáticas)
- Knowledge expansion
- News update
- Coverage audit

## Tareas semanales
- Revisar PRs automáticos
- Verificar links rotos
- Actualizar dependencias

## Tareas mensuales
- Revisar métricas de cobertura
- Actualizar roadmap
- Limpiar contenido obsoleto
- Revisar issues pendientes

## Comandos útiles
```bash
make validate    # Validar estructura
make audit       # Auditoría
make check-links # Verificar links
make clean       # Limpiar temporales
```
