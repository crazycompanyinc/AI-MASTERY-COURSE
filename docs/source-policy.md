# Política de Fuentes

## Principios
1. No copiar contenido protegido en masa
2. Crear explicaciones originales
3. Citar fuentes con enlaces
4. Respetar licencias
5. Preferir fuentes académicas y oficiales

## Fuentes preferidas
- arXiv, Papers With Code, Semantic Scholar
- Documentación oficial de frameworks
- Blogs de investigación (OpenAI, Anthropic, DeepMind)
- Cursos abiertos (MIT, Stanford, Berkeley)

## Fuentes a evitar
- Contenido detrás de paywall
- Scraping abusivo
- Fuentes de baja calidad
- Contenido sin verificar

## Citación
Siempre incluir:
- Autor(es)
- Título
- Fecha
- URL
