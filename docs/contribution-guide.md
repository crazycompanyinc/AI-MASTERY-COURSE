# Guía de Contribución

## Formas de contribuir
- Añadir lecciones
- Mejorar explicaciones
- Añadir papers
- Corregir errores
- Traducir contenido
- Reportar issues

## Proceso
1. Fork
2. Branch: `git checkout -b feature/mi-cambio`
3. Cambios + `make validate`
4. Commit + Push
5. Pull Request

## Estándares
- Markdown para docs
- Python 3.10+ para código
- Español como idioma principal
- Citar fuentes
- Contenido original
