# Lección: Natural Language Processing

## Objetivo
Comprender las técnicas de NLP desde los métodos clásicos hasta los LLMs modernos.

## Contenido

### Fundamentos
- Tokenización, stemming, lematización
- Bag-of-words, TF-IDF
- N-grams
- Word embeddings (Word2Vec, GloVe, FastText)

### Tareas de NLP
- Clasificación de texto
- Named Entity Recognition (NER)
- Part-of-Speech tagging
- Sentiment analysis
- Machine translation
- Question answering
- Summarization
- Text generation

### Arquitecturas
- RNNs/LSTMs para texto
- Seq2seq con atención
- Transformers (BERT, GPT, T5)
- LLMs (fine-tuning, prompting)

### Embeddings
- Word2Vec (Skip-gram, CBOW)
- GloVe
- FastText
- Contextual embeddings (ELMo, BERT)
- Sentence embeddings (Sentence-BERT)

### Aplicaciones
- Chatbots
- Búsqueda semántica
- Extracción de información
- Generación de texto
- Traducción automática

## Referencias
- Jurafsky & Martin. Speech and Language Processing.
- Hugging Face NLP course: https://huggingface.co/learn/nlp-course
