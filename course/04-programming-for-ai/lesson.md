# Lección: Programación para IA

## Objetivo
Dominar Python y las herramientas de programación esenciales para trabajar con IA.

## Contexto
Python es el lenguaje dominante en IA. Librerías como NumPy, Pandas, scikit-learn, PyTorch y TensorFlow son el estándar de la industria.

## Contenido

### Python para IA
- Variables, tipos de datos, control de estructuras
- Funciones, clases, módulos
- List comprehensions, generators, decorators
- Manejo de errores y excepciones
- Virtual environments y gestión de paquetes

### NumPy
- Arrays multidimensionales
- Broadcasting
- Operaciones vectorizadas
- Álgebra lineal con NumPy

### Pandas
- DataFrames y Series
- Limpieza de datos
- Groupby, merge, pivot
- Time series

### Estructuras de datos
- Listas, diccionarios, sets, tuplas
- Stacks, queues, heaps
- Árboles y grafos

### Algoritmos
- Búsqueda y ordenamiento
- Programación dinámica
- Algoritmos de grafos (BFS, DFS, Dijkstra)
- Complejidad computacional (Big-O)

### Buenas prácticas
- Código limpio y legible
- Testing (pytest)
- Documentación
- Git y control de versiones
- Entornos virtuales
