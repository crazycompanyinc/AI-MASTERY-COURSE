# Lección: Fundamentos de Datos

## Objetivo
Comprender cómo trabajar con datos: recolección, limpieza, transformación y preparación para ML.

## Contenido

### Tipos de datos
- Estructurados (tablas, SQL)
- No estructurados (texto, imágenes, audio)
- Semi-estructurados (JSON, XML)
- Time series
- Datos geoespaciales

### Fuentes de datos
- APIs, web scraping, bases de datos
- Datasets públicos (Kaggle, UCI, Google Dataset Search)
- Datos sintéticos

### Limpieza de datos
- Valores faltantes (imputación, eliminación)
- Outliers (detección, tratamiento)
- Duplicados
- Inconsistencias
- Formato y tipos

### Feature engineering
- Escalado (StandardScaler, MinMaxScaler)
- Encoding (one-hot, label, target)
- Binning
- Features polinómicos
- Features temporales
- Text features (TF-IDF, n-grams)

### Exploratory Data Analysis (EDA)
- Estadísticas descriptivas
- Visualización (matplotlib, seaborn)
- Correlaciones
- Distribuciones

### Data pipelines
- ETL (Extract, Transform, Load)
- Apache Airflow
- Prefect
- DVC (Data Version Control)

- pandas docs: https://pandas.pydata.org/docs/
- scikit-learn preprocessing: https://scikit-learn.org/stable/modules/preprocessing.html
