# Lección: Embeddings y Bases Vectoriales

## Objetivo
Comprender cómo funcionan los embeddings y las bases de datos vectoriales.

## Contenido

### Embeddings
- Word2Vec, GloVe, FastText
- Sentence embeddings (Sentence-BERT)
- Embeddings multimodales (CLIP)
- OpenAI embeddings, Cohere embeddings

### Búsqueda vectorial
- Similitud coseno
- Distancia euclidiana
- ANN (Approximate Nearest Neighbors)

### Bases de datos vectoriales
- FAISS (Facebook)
- Chroma (open-source)
- Qdrant (Rust)
- Weaviate
- Pinecone (managed)
- pgvector (PostgreSQL)

### Indexación
- HNSW (Hierarchical Navigable Small World)
- IVF (Inverted File Index)
- PQ (Product Quantization)
- ScaNN

### Aplicaciones
- Búsqueda semántica
- RAG (Retrieval-Augmented Generation)
- Recomendación
- Deduplicación
- Clustering
