# Lección: Machine Learning

## Objetivo
Comprender los fundamentos del Machine Learning: tipos de aprendizaje, algoritmos principales y evaluación.

## Tipos de aprendizaje
- **Supervisado**: datos etiquetados (clasificación, regresión)
- **No supervisado**: sin etiquetas (clustering, reducción de dimensionalidad)
- **Refuerzo**: aprendizaje por interacción
- **Self-supervisado**: el modelo genera sus propias etiquetas

## Algoritmos principales
- Regresión lineal y logística
- KNN, Árboles de decisión
- Random Forest, XGBoost
- SVM, Naive Bayes

## Evaluación
- Accuracy, Precision, Recall, F1
- Cross-validation
- Overfitting vs Underfitting
- Bias-variance tradeoff

## Pipeline de ML
1. Definir problema
2. Recolectar datos
3. EDA
4. Preprocesar
5. Entrenar
6. Evaluar
7. Tunear
8. Desplegar
