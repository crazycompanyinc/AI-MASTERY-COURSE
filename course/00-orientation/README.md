# Módulo 00: Orientación

## Bienvenido a AI-MASTERY-COURSE

Este módulo te explica cómo navegar, usar y sacar el máximo provecho de esta biblioteca viva de Inteligencia Artificial.

## Cómo está organizado el curso

El curso está dividido en **30 módulos temáticos** que cubren todo el espectro de la IA, desde la historia y filosofía hasta investigación frontier. Cada módulo contiene:

- **lesson.md** — Contenido principal de la lección
- **summary.md** — Resumen ejecutivo
- **key-concepts.md** — Conceptos clave definidos
- **examples.md** — Ejemplos prácticos
- **exercises.md** — Ejercicios para practicar
- **quiz.md** — Preguntas de autoevaluación
- **projects.md** — Proyectos asociados
- **further-reading.md** — Lecturas recomendadas
- **references.md** — Fuentes y citas
- **practical-lab.md** — Laboratorio práctico

## Cómo estudiar cada módulo

1. **Lee el summary.md** — Obtén la visión general en 5 minutos
2. **Estudia lesson.md** — Contenido profundo con explicaciones
3. **Revisa key-concepts.md** — Asegúrate de entender los términos
4. **Haz los exercises.md** — Practica lo aprendido
5. **Completa el practical-lab.md** — Código y experimentos
6. **Haz el quiz.md** — Autoevaluación
7. **Explora further-reading.md** — Profundiza
8. **Construye los projects.md** — Aplica todo

- Python 3.10+
- 8GB RAM mínimo (16GB recomendado)

## Instalación rápida

```bash
git clone https://github.com/YOUR_USERNAME/AI-MASTERY-COURSE.git
cd AI-MASTERY-COURSE
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
make validate
```
