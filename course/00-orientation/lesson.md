# Lección: Orientación al Curso

## Objetivo
Entender la estructura, filosofía y rutas de aprendizaje de AI-MASTERY-COURSE.

## Contexto
Antes de sumergirnos en IA, necesitas un mapa. Este curso tiene 30 módulos, cientos de lecciones, laboratorios, proyectos y un sistema de actualización automática. Sin un mapa, te perderás.

## Explicación principal

### ¿Qué es AI-MASTERY-COURSE?

Es una **biblioteca viva** de Inteligencia Artificial. No es un curso tradicional con un final. Es un recurso que crece y se actualiza diariamente mediante automatización.

### Filosofía del diseño

1. **Progresivo**: De simple a complejo
2. **Práctico**: Código real, no solo teoría
3. **Completo**: Cubre todo el espectro de IA
4. **Actualizado**: Sistema de actualización diaria
5. **Navegable**: Múltiples rutas de aprendizaje
6. **Abierto**: Cualquiera puede contribuir
7. **Verificable**: Auditorías de cobertura

### Estructura de cada módulo

Cada módulo del curso sigue una estructura pedagógica consistente:

1. **README.md** — Visión general del módulo
2. **lesson.md** — Contenido principal (la lección)
3. **summary.md** — Resumen ejecutivo (5 minutos)
4. **key-concepts.md** — Glosario del módulo
5. **examples.md** — Ejemplos prácticos
6. **exercises.md** — Ejercicios
7. **quiz.md** — Autoevaluación
8. **projects.md** — Proyectos
9. **further-reading.md** — Lecturas adicionales
10. **references.md** — Fuentes
11. **practical-lab.md** — Laboratorio práctico

### Sistema de actualización automática

El curso se actualiza diariamente mediante:

- **GitHub Actions**: 5 workflows automatizados
- **Cron local**: 3 tareas programadas
- **Scripts de Python**: Recolección, clasificación y escritura de contenido

Cada día:
1. Se buscan nuevos papers en arXiv
2. Se leen noticias y blogs técnicos
3. Se actualiza el trend-watch
4. Se audita la cobertura
5. Se generan PRs automáticos

### Métrica de calidad: AI_LIBRARY_TOTALITY_SCORE

El curso se evalúa con una métrica propia que considera:
- **Breadth** (Amplitud): ¿Cuántos temas cubre?
- **Depth** (Profundidad): ¿Qué tan profundo llega?
- **Practicality** (Practicidad): ¿Se puede aplicar?
- **Academic grounding** (Base académica): ¿Está bien fundamentado?
- **Freshness** (Actualidad): ¿Está actualizado?
- **Navigability** (Navegabilidad): ¿Es fácil de navegar?
- **Originality** (Originalidad): ¿Es contenido original?
- **Reproducibility** (Reproducibilidad): ¿Se puede reproducir?
- **Pedagogical quality** (Calidad pedagógica): ¿Enseña bien?
- **Advanced coverage** (Cobertura avanzada): ¿Llega a temas frontier?
- **Update velocity** (Velocidad de actualización): ¿Qué tan rápido crece?

## Conceptos clave

- **Biblioteca viva**: Un recurso educativo que crece y se actualiza automáticamente
- **Ruta de aprendizaje**: Un camino curado a través del curso para un perfil específico
- **Módulo**: Una unidad temática del curso
- **Laboratorio**: Un ejercicio práctico con código
- **Proyecto**: Una aplicación integradora
- **Auditoría de cobertura**: Evaluación sistemática de qué temas están cubiertos y cuáles faltan

## Errores comunes

1. **Intentar leer todo linealmente**: Usa las rutas de aprendizaje
2. **Saltar los ejercicios**: La práctica es esencial
3. **Ignorar los prerequisitos**: Cada módulo asume conocimiento previo
4. **No ejecutar los labs**: El código debe ejecutarse, no solo leerse
5. **Olvidar las actualizaciones**: El curso cambia diariamente

## Aplicaciones reales

Este curso está diseñado para:
- Autodidactas que quieren dominar IA
- Universidades que buscan material complementario
- Empresas que entrenan equipos
- Bootcamps que necesitan currículo
- Investigadores que buscan referencia rápida

## Limitaciones

- El curso es tan bueno como sus contribuidores
- Los labs requieren hardware para algunos ejercicios avanzados
- Las APIs opcionales pueden tener costos
- El contenido en español puede tener menos recursos que en inglés

## Relación con otros módulos

Este módulo es el punto de entrada a todos los demás. Cada módulo posterior asume que entiendes la estructura y filosofía explicada aquí.

## Ejercicios

1. **Elige tu ruta de aprendizaje** basándote en tu perfil y objetivos
2. **Clona el repositorio** y ejecuta `make validate`
3. **Navega la estructura** de al menos 3 módulos
4. **Lee el summary.md** de tu primer módulo
5. **Ejecuta un lab** de los recomendados en tu ruta

3. ¿Cómo se actualiza el curso automáticamente?
4. ¿Qué es el AI_LIBRARY_TOTALITY_SCORE?
